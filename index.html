Meta Browser Warning

-This is a basic HTML & Javascript application that will detect & warn the user if the site is being viewed with Facebook's or Instagram's browser.

-The warning tells the user that this website should not be viewed in the current browser & that the user should open the page in a compatible browser.

-I made this basic application because Meta's browsers, which are used when you open a link in one of their messager apps, can make certain websites look different depending on the CSS used.
